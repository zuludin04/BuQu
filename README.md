# BuQu

BuQu is a visual space for your thoughts and findings from the books you read. Every time we read a book, we encounter meaningful quotes or sentences that we want to save. Instead of just list-based storage, BuQu allows you to organize these insights into interactive visual boards, helping you see the connections between ideas.

With BuQu, you can save quotes along with metadata like page numbers, book titles, authors, and categories. The application leverages modern features like speech-to-text and image-to-text to make capturing long quotes effortless.

## Features

- **Visual Board Editor**: An infinite canvas to organize your thoughts and book insights visually.
- **Connect Ideas**: Link related notes and quotes using "ropes" with customizable colors and captions.
- **Multimedia Note Cards**: Create cards for text, quotes, images, or specific books.
- **Smart Alignment**: Automatically tidy up and align your selected notes for a cleaner board layout.
- **Speech-to-Text & Image-to-Text**: Quickly input long passages using voice recognition or OCR.
- **Collection Import**: Easily import your saved books and quotes directly into your visual boards.
- **Categorization**: Organize your collection with customizable categories.
- **Share Quotes**: Share your favorite findings with friends and family.
